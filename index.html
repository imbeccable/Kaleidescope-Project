This program utlizes WebGl by creating triangles at random points duplicated, flipped, and transposed a certain number of times about an origin point to create a kaleidescopic effect. The number of slices to this image is depending first on a randomly generated number with a base line number of 4, and then can be altered by user input via a slider. The number of shapes within each section is given an initial value and then can be determined based on user input via a slider, with a range from 4 to 20.
